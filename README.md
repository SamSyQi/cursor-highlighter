# Cursor Highlighter (Open Source)

A lightweight, customizable **cursor highlighter for Windows**, designed to improve visibility during screen recording, presentations, and live demos.

Built with Python and Tkinter, this application runs as a transparent overlay above all windows while remaining fully click-through.
